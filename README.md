# Session Logger - Приложение для логирования НРИ-сессий

## Описание
Приложение для удобного логированния любых НРИ сессий. Оно пригодится как игрокам (Например для Adventures league D&D), так и мастерам для их увлекательных кампаний.

## Быстрый старт
### Требования
- .NET MAUI SDK
- IDE (Visual Studio 2022 / Rider)
- CommunityToolkit.MVVM
- LiteDB

## РЕКОМЕНДАЦИИ
Перед началом разработки настоятельно рекомендую изучить следующее
- Что такое MVVM
- Что такое NoSQL
- Что такое паттерны проектирования
- конкретные паттерны *стратегия(strategy)*, *одиночка(singletone)*, *фабрика(factory)*, *наблюдатель(observer)*, *команда(command)*, *состояние(state)*, *строитель(builder)*, *посредник(mediator)*, *посетитель(visitor)*
- Что такое DEPENDENCY INJECTION (DI)
- Что такое Bson
Также необходимо ознакомиться с соглашениями написания кода, которые хранятся в Documentations/For Contributions
