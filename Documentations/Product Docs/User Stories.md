## EPIC 1: Управление кампаниями

### US-001: Создание новой кампании
| Поле | Значение |
|------|----------|
| **ID** | US-001 |
| **Название** | Создание новой кампании |
| **Как** | Пользователь (Мастер или Игрок) |

| **Чтобы** | Начать ведение логов для новой игровой группы |

- [ ] Кнопка "Создать кампанию" доступна с главного экрана
