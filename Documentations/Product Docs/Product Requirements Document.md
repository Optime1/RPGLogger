# Product Requirements Document

## 1. Vision
Приложение для быстрого и удобного логирования сессий НРИ в оффлайн-режиме

## 2. Problem Statement
Мастера и игроки ведут записи на бумаге/в заметках, что:
- Неудобно искать информацию
- Теряются записи
- Сложно делиться с другими игроками
- Нет структуры

## 3. Target Audience
- Мастера НРИ (D&D, Pathfinder, GURPS и др.)
- Игроки, ведущие заметки по кампаниям
- Возраст: 16-45 лет
- Tech-savvy: средний уровень

## 4. Success Metrics
- Время создания лога сессии < 2 минут
- Retention D7 > 40%
- Rating в магазинах > 4.5
